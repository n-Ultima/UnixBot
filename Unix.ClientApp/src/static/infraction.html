Infraction Commands

Parameters are marked as optional if surrounded by ( ).

These commands require you to have Moderator permissions within your guild's configuration, or be the guild owner.


infractions [user | userId] - Fetches the infractions for the user provided in that guild.

infraction [id] - Fetches information for the infraction ID provided.

infraction-update [id] [reason] - Updates the infraction provided with the new reason.

infraction-delete [id] [reason] - Deletes the infraction provided.
