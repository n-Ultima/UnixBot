Moderation Commands

Parameters are marked as optional if surrounded by ( ).

These commands require you to have Moderator permissions within your guild's configuration, or be the guild owner.


purge [count] (user) - Purges the amount of message provided from the channel. If a user is provided, only their messages are deleted.

note [user] [content] - Records a note for the user provided.

warn [user] [reason] - Warns a user for the provided reason.

kick [user] [reason] - Kicks the user provided from the guild.

ban [user] (duration) [reason] - Bans the user provided. If a duration is provided, the user will be unbanned after said duration has passed.

mute [user] [reason] [duration] - Mutes the user provided. They will be unmuted after the duration provided has passed.

unmute [user] [reason] - Unmutes the user provided.

unban [userId] [reason] - Unbans the user provided.
